## Enabled

This plugin is enabled when there is match in `dependencies` or
`devDependencies`:

- `postcss`

- `next`

## Default configuration

```json title="knip.json"
{
  "postcss": {
    "config": ["postcss.config.{cjs,js}", "postcss.config.json", "package.json"]
  }
}
```
