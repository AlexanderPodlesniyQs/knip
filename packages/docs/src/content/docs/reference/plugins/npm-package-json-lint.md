## Enabled

This plugin is enabled when there is match in `dependencies` or
`devDependencies`:

- `npm-package-json-lint`

## Default configuration

```json title="knip.json"
{
  "npm-package-json-lint": {
    "config": [
      ".npmpackagejsonlintrc.json",
      "npmpackagejsonlint.config.js",
      "package.json"
    ]
  }
}
```
